Migrar a aplicação “CAIXA Loterias — Indicadores Estratégicos” para PHP, preservando integralmente o banco SQLite, a estrutura visual, o layout, os dados gravados, as regras de cálculo, os fluxos de lançamento, homologação, relatórios e todas as funcionalidades já implementadas.

CONTEXTO DO PROJETO

A aplicação atual foi desenvolvida para validação local do sistema de Indicadores Estratégicos da CAIXA Loterias.

Agora precisamos migrar a aplicação para PHP, pois PHP é a linguagem aceita no ambiente/servidor da CAIXA Loterias.

O banco atual em SQLite deve ser preservado.

A aplicação deve continuar funcionando localmente com SQLite, mas com backend em PHP.

1. PRESERVAÇÃO DO BANCO SQLITE

O banco existente deve ser mantido em:

/database/indicadores.sqlite

Antes de qualquer alteração, criar backup automático:

/database/backups/indicadores-backup-ANTES-MIGRACAO-PHP-AAAA-MM-DD-HHMM.sqlite

Também criar backup JSON, se a aplicação já possuir exportador.

A aplicação PHP deve usar esse arquivo SQLite como fonte oficial local.

No PHP, conectar usando PDO:

sqlite:../database/indicadores.sqlite

ou caminho absoluto seguro definido em config.php.

Exemplo de configuração:

define('DB_PATH', __DIR__ . '/../../database/indicadores.sqlite');

2. NÃO RECRIAR TABELAS SE JÁ EXISTIREM

Ao iniciar a aplicação, verificar se o banco existe.

Se existir:
- não sobrescrever;
- não apagar;
- não recriar;
- apenas conectar e usar.

Se não existir:
- criar a partir do schema.sql somente se for uma instalação nova.

Regra:

if file_exists('/database/indicadores.sqlite'):
    usar banco existente
else:
    criar novo banco com schema.sql

3. CAMADA DE CONEXÃO PHP

Criar classe:

/app/core/Database.php

Responsabilidades:

- abrir conexão PDO com SQLite;
- ativar modo de erro por exceção;
- ativar foreign keys;
- garantir UTF-8;
- fornecer conexão para repositories.

Exemplo esperado:

class Database {
    private static $connection = null;

    public static function getConnection(): PDO {
        if (self::$connection === null) {
            $pdo = new PDO('sqlite:' . DB_PATH);
            $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
            $pdo->exec('PRAGMA foreign_keys = ON');
            self::$connection = $pdo;
        }

        return self::$connection;
    }
}
